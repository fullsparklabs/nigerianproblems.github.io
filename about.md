<div class="archive-intro">
  <h1>About Nigerian Problems Archive</h1>
  <p>This project aims to create a comprehensive, searchable archive of problems that Nigeria has faced from independence in 1960 to the present day.</p>
</div>

<div class="timeline">
  <h2>What This Archive Contains</h2>
  <p>The Nigerian Problems Archive documents major challenges, crises, and issues that have affected Nigeria and its citizens over the decades. This includes:</p>
  
  <ul style="margin: 1rem 0 1rem 2rem; line-height: 2;">
    <li>Political instability and military coups</li>
    <li>Economic challenges and recessions</li>
    <li>Security issues and conflicts</li>
    <li>Infrastructure deficits</li>
    <li>Public health crises</li>
    <li>Educational challenges</li>
    <li>Environmental problems</li>
    <li>Corruption and governance issues</li>
    <li>Social and ethnic tensions</li>
    <li>And many more...</li>
  </ul>

  <h2>How It Works</h2>
  <p>This blog is automatically generated using:</p>
  
  <ol style="margin: 1rem 0 1rem 2rem; line-height: 2;">
    <li><strong>Web Scraping:</strong> Automated scripts gather news and information from multiple sources</li>
    <li><strong>AI Analysis:</strong> Ollama-powered AI (Qwen model) processes, categorizes, and summarizes content</li>
    <li><strong>Jekyll Static Site:</strong> Content is formatted as a Jekyll blog for easy GitHub Pages hosting</li>
    <li><strong>GitHub Actions:</strong> Automated workflows run the scraping and site generation on a schedule</li>
  </ol>

  <h2>Timeline Coverage</h2>
  <p>The archive covers problems from <strong>1960</strong> (Nigerian independence) to the present day, organized by year and category for easy navigation.</p>

  <h2>Contribute</h2>
  <p>This is an open-source project. If you'd like to contribute or improve the scraping and categorization, please visit our GitHub repository.</p>
</div>
